# Purple Times

O Purple Times é um site de notícias desenvolvido como projeto escolar, com foco em oferecer informação rápida, organizada e visualmente agradável.
O site inclui categorias, notícias destacadas, barra de pesquisa, sidebar dinâmica e páginas individuais para cada notícia.

---

## Funcionalidades

- Card de notícia principal  
- Lista de notícias (estáticas ou via API)  
- Sidebar com categorias  
- Sessão de “Mais Lidas”  

---
